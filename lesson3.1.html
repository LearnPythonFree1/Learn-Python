Numeric Types

What are numeric types in Python?

Definition

Python has two main numeric types: integers (int) and floating-point numbers (float). Integers are whole numbers, while floats are numbers with decimal points.

Explanation

Think of integers like counting whole items (1, 2, 3) and floats like measuring precise amounts (1.5, 2.75). Python automatically handles the type based on how you write the number.

# Integers
age = 25
count = -10
big_number = 1000000

# Floats
height = 1.75
temperature = -3.5
pi = 3.14159

# Type checking
print(type(age))      # 
print(type(height))   #

This code demonstrates different ways to create integers and floats, and how to check their types.

Numeric Operations

How to perform calculations?

Definition

Python supports all basic arithmetic operations: addition (+), subtraction (-), multiplication (*), division (/), integer division (//), modulo (%), and exponentiation (**).

Explanation

Think of these operations like a calculator - you can add, subtract, multiply, and divide numbers just like you would on a calculator, but with more advanced features.

# Basic arithmetic
a = 10
b = 3

print(a + b)    # Addition: 13
print(a - b)    # Subtraction: 7
print(a * b)    # Multiplication: 30
print(a / b)    # Division: 3.3333...
print(a // b)   # Integer division: 3
print(a % b)    # Modulo (remainder): 1
print(a ** b)   # Exponentiation: 1000

This code shows all the basic arithmetic operations available in Python.

String Basics

What are strings?

Definition

A string is a sequence of characters enclosed in quotes (single or double). Strings are immutable, meaning once created, their individual characters cannot be changed.

Explanation

Think of a string like a sentence - it's a collection of letters, numbers, and symbols that form words and phrases. Just like you can't change individual letters in a printed book, you can't change individual characters in a string.

# String creation
name = "John"
message = 'Hello, World!'
multiline = """This is a
multi-line string"""

# String operations
print(name + " " + message)  # Concatenation
print(name * 3)              # Repetition
print(len(name))             # Length
print(name[0])               # Indexing
print(name[1:3])             # Slicing

This code demonstrates different ways to create and manipulate strings.

String Methods

How to manipulate strings?

Definition

String methods are built-in functions that perform operations on strings. They return a new string rather than modifying the original string.

Explanation

Think of string methods like text editing tools - they help you transform text in various ways, such as changing case, removing spaces, or replacing parts of the text.

text = "  Python Programming  "

# Common string methods
print(text.strip())          # Remove spaces
print(text.lower())          # Convert to lowercase
print(text.upper())          # Convert to uppercase
print(text.replace("Python", "Java"))  # Replace text
print(text.split())          # Split into words
print(text.count("o"))       # Count occurrences
print(text.find("Pro"))      # Find substring

This code shows some of the most commonly used string methods in Python.

String Formatting

How to format strings?

Definition

String formatting allows you to create strings by inserting values into a template. Python provides multiple ways to format strings: f-strings, .format(), and % operator.

Explanation

Think of string formatting like filling in a form - you have a template with placeholders, and you fill in the values where needed.

name = "Alice"
age = 25
score = 95.5

# f-strings (recommended)
print(f"{name} is {age} years old")
print(f"Score: {score:.1f}%")

# .format() method
print("{} is {} years old".format(name, age))
print("Score: {:.1f}%".format(score))

# % operator (old style)
print("%s is %d years old" % (name, age))
print("Score: %.1f%%" % score)

This code demonstrates different ways to format strings in Python.

Type Conversion

How to convert between types?

Definition

Type conversion (casting) is the process of converting a value from one data type to another. Python provides built-in functions for type conversion: int(), float(), and str().

Explanation

Think of type conversion like translating between languages - you're taking information in one format and converting it to another format that's more suitable for your needs.

# String to number
age_str = "25"
age_int = int(age_str)
height_str = "1.75"
height_float = float(height_str)

# Number to string
age = 25
age_str = str(age)
pi = 3.14159
pi_str = str(pi)

# Float to int (truncates decimal)
price = 19.99
price_int = int(price)  # 19

print(f"Age as string: {age_str}")
print(f"Height as float: {height_float}")
print(f"Price as integer: {price_int}")

This code shows how to convert between different data types in Python.

Practice Exercise: Calculator

Let's put it all together!

Definition

A simple calculator program that combines numeric operations, string formatting, and type conversion to create an interactive calculation tool.

Explanation

This exercise combines all the concepts we've learned to create a functional calculator. You'll use numeric operations for calculations, string formatting for output, and type conversion for user input.

def calculator():
    print("Simple Calculator")
    print("1. Add")
    print("2. Subtract")
    print("3. Multiply")
    print("4. Divide")
    
    choice = input("Enter choice (1-4): ")
    num1 = float(input("Enter first number: "))
    num2 = float(input("Enter second number: "))
    
    if choice == "1":
        result = num1 + num2
        print(f"{num1} + {num2} = {result}")
    elif choice == "2":
        result = num1 - num2
        print(f"{num1} - {num2} = {result}")
    elif choice == "3":
        result = num1 * num2
        print(f"{num1} * {num2} = {result}")
    elif choice == "4":
        if num2 != 0:
            result = num1 / num2
            print(f"{num1} / {num2} = {result:.2f}")
        else:
            print("Error: Cannot divide by zero!")
    else:
        print("Invalid choice!")

calculator()

This exercise combines numeric operations, string formatting, and type conversion to create an interactive calculator.
