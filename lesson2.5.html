While Loops

Learn how to use while loops to repeat code while a condition is true.

15 minutes Beginner Control Flow

Basic While Loop

What is a while loop?

Definition

A while loop is a control flow statement that allows code to be executed repeatedly based on a given boolean condition. The loop continues as long as the condition remains true.

Explanation

Think of a while loop like a repeating task that continues until a specific condition is met. For example, counting down from 5 to 1 is like a while loop - you keep counting until you reach 1.

count = 1
while count <= 5:
    print(f"Count is: {count}")
    count += 1

This loop will print numbers from 1 to 5.

While Loop with Break

How to exit a while loop?

Definition

The break statement is used to exit a loop prematurely when a specific condition is met, regardless of the loop's original condition.

Explanation

Break is like an emergency exit in a building - when you need to leave immediately, you use it regardless of the normal exit procedures.

while True:
    number = int(input("Enter a number (0 to exit): "))
    if number == 0:
        break
    print(f"You entered: {number}")

This loop continues until the user enters 0.

While Loop with Continue

How to skip iterations?

Definition

The continue statement skips the rest of the current iteration and jumps to the next iteration of the loop.

Explanation

Continue is like skipping a step in a process - you move on to the next step without completing the current one.

count = 0
while count < 5:
    count += 1
    if count == 3:
        continue
    print(f"Count is: {count}")

This loop skips printing when count is 3.

While Loop with Else

What happens after the loop?

Definition

The else clause in a while loop executes when the loop condition becomes false, but only if the loop wasn't terminated by a break statement.

Explanation

Think of it like a completion message - if you finish a task normally, you get a completion message, but if you quit early, you don't.

count = 1
while count <= 3:
    print(f"Count is: {count}")
    count += 1
else:
    print("Loop finished!")

The else clause runs after the loop completes normally.

Nested While Loops

How to use loops inside loops?

Definition

A nested while loop is a while loop inside another while loop, creating a hierarchical structure of iterations.

Explanation

Think of it like a grid - the outer loop moves through rows, and the inner loop moves through columns.

i = 1
while i <= 3:
    j = 1
    while j <= 3:
        print(f"i={i}, j={j}")
        j += 1
    i += 1

This creates a 3x3 grid of coordinates.

While Loop with List

How to process lists with while?

Definition

Using a while loop to iterate through list elements by maintaining an index counter.

Explanation

It's like reading a book page by page - you keep track of which page you're on and stop when you reach the end.

fruits = ["apple", "banana", "cherry"]
i = 0
while i < len(fruits):
    print(fruits[i])
    i += 1

This prints each fruit in the list.

While Loop with User Input

How to handle user input in loops?

Definition

Using while loops to repeatedly collect and process user input until a specific condition is met.

Explanation

It's like taking a survey - you keep asking questions until you have enough responses.

numbers = []
while len(numbers) < 3:
    num = int(input("Enter a number: "))
    numbers.append(num)
print(f"Sum: {sum(numbers)}")

This collects 3 numbers and calculates their sum.

While Loop with String

How to process strings with while?

Definition

Using a while loop to iterate through string characters by maintaining an index counter.

Explanation

It's like reading a word letter by letter - you process each character until you reach the end of the word.

text = "Python"
i = 0
while i < len(text):
    print(text[i])
    i += 1

This prints each character in the string.

While Loop with Dictionary

How to process dictionaries with while?

Definition

Using a while loop to iterate through dictionary items by maintaining an index counter for the keys.

Explanation

It's like going through a dictionary word by word - you look up each word and its definition.

person = {"name": "John", "age": 30, "city": "New York"}
keys = list(person.keys())
i = 0
while i < len(keys):
    key = keys[i]
    print(f"{key}: {person[key]}")
    i += 1

This prints each key-value pair in the dictionary.

Practice Exercise: Number Guessing Game

Let's put it all together!

Definition

A number guessing game that combines while loops, user input, and conditional statements to create an interactive experience.

Explanation

This exercise combines all the concepts we've learned to create a complete game. You'll use while loops for the game loop, user input for guesses, and conditional statements for feedback.

import random

def play_game():
    number = random.randint(1, 100)
    attempts = 0
    
    while True:
        guess = int(input("Guess the number (1-100): "))
        attempts += 1
        
        if guess < number:
            print("Too low!")
        elif guess > number:
            print("Too high!")
        else:
            print(f"Correct! You took {attempts} attempts.")
            break

while True:
    play_game()
    if input("Play again? (y/n): ").lower() != 'y':
        break

This exercise combines while loops, if statements, and user input to create an interactive game.
