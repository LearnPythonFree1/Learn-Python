Lesson 4: Operators in Python
Learn about the different types of operators in Python and how to use them in your programs. This lesson covers arithmetic, comparison, logical, assignment, and membership/identity operators with practical examples.
25 min
5 sections
Question 1
What are arithmetic operators in Python and how are they used?
Answer
Arithmetic operators are used to perform mathematical operations: - Addition: + (e.g., 3 + 2 → 5) - Subtraction: - (e.g., 5 - 2 → 3) - Multiplication: * (e.g., 4 * 2 → 8) - Division: / (e.g., 8 / 2 → 4.0) - Floor Division: // (e.g., 7 // 2 → 3) - Modulus: % (e.g., 7 % 2 → 1) - Exponentiation: ** (e.g., 2 ** 3 → 8)
Definition
Arithmetic operators allow you to perform basic mathematical calculations in Python. They work with numbers (integers and floats) and are fundamental for any computation.
Question 2
What are comparison operators in Python and what do they return?
Answer
Comparison operators compare two values and return a Boolean result (True or False): - Equal to: == - Not equal to: != - Greater than: > - Less than: < - Greater than or equal to: >= - Less than or equal to: <= Example: 5 > 3 returns True 2 == 4 returns False
Definition
Comparison operators are used to compare values. They are essential for decision-making in code, such as in if statements and loops.
Question 3
How do logical operators work in Python?
Answer
Logical operators are used to combine conditional statements: - and: True if both operands are True - or: True if at least one operand is True - not: Inverts the Boolean value Example: (5 > 3) and (2 < 4) returns True not (5 == 3) returns True
Definition
Logical operators allow you to build complex conditions by combining multiple Boolean expressions. They are crucial for flow control in programs.
Question 4
What are assignment operators in Python and how are they used?
Answer
Assignment operators are used to assign values to variables: - = : Assigns value (e.g., x = 5) - += : Adds and assigns (e.g., x += 2 is x = x + 2) - -= : Subtracts and assigns - *= : Multiplies and assigns - /= : Divides and assigns - //= : Floor divides and assigns - %= : Modulus and assigns - **= : Exponentiates and assigns
Definition
Assignment operators combine assignment with another operation. They make code more concise and readable when updating variable values.
Question 5
What are membership and identity operators in Python?
Answer
Membership operators: - in: True if a value is found in a sequence (e.g., 3 in [1,2,3] → True) - not in: True if a value is not found in a sequence Identity operators: - is: True if two variables point to the same object - is not: True if they do not Example: a = [1,2,3]; b = a; a is b → True a is not [1,2,3] → True
Definition
Membership operators test for presence in a sequence. Identity operators check if two references point to the same object in memory. Both are useful for comparisons beyond just value equality.
