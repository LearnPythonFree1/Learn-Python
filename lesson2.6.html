For Loops

Learn how to use for loops to iterate over sequences and collections.

Basic For Loop

What is a for loop?

Definition

A for loop is a control flow statement that iterates over a sequence (like a list, tuple, or string) and executes a block of code for each element in the sequence.

Explanation

Think of a for loop like going through a shopping list - you check each item one by one until you've gone through the entire list.

fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    print(fruit)

This loop prints each fruit in the list.

For Loop with Range

How to loop through numbers?

Definition

The range() function generates a sequence of numbers, and for loops can iterate over this sequence to execute code a specific number of times.

Explanation

It's like counting - you start at one number and count up to another number, doing something at each step.

for i in range(5):
    print(f"Number: {i}")

This loop prints numbers from 0 to 4.

For Loop with Break

How to exit a for loop?

Definition

The break statement in a for loop allows you to exit the loop prematurely when a specific condition is met.

Explanation

Break is like finding what you're looking for in a search - once you find it, you can stop looking through the rest of the items.

for i in range(10):
    if i == 5:
        break
    print(i)

This loop stops when it reaches 5.

For Loop with Continue

How to skip iterations?

Definition

The continue statement in a for loop skips the current iteration and moves to the next element in the sequence.

Explanation

Continue is like skipping a step in a process - you move on to the next step without completing the current one.

for i in range(5):
    if i == 2:
        continue
    print(i)

This loop skips printing when i is 2.

For Loop with Else

What happens after the loop?

Definition

The else clause in a for loop executes when the loop has exhausted all items in the sequence, but only if the loop wasn't terminated by a break statement.

Explanation

Think of it like a completion message - if you finish going through all items normally, you get a completion message, but if you quit early, you don't.

for i in range(3):
    print(i)
else:
    print("Loop finished!")

The else clause runs after the loop completes normally.

Nested For Loops

How to use loops inside loops?

Definition

A nested for loop is a for loop inside another for loop, creating a hierarchical structure of iterations.

Explanation

Think of it like a grid - the outer loop moves through rows, and the inner loop moves through columns.

for i in range(3):
    for j in range(3):
        print(f"i={i}, j={j}")

This creates a 3x3 grid of coordinates.

For Loop with Enumerate

How to get both index and value?

Definition

The enumerate() function adds a counter to an iterable and returns it as an enumerate object, allowing you to access both the index and value in a for loop.

Explanation

It's like having a numbered list - you can see both the number and the item at the same time.

fruits = ["apple", "banana", "cherry"]
for index, fruit in enumerate(fruits):
    print(f"{index}: {fruit}")

This prints each fruit with its index.

For Loop with Zip

How to loop through multiple lists?

Definition

The zip() function combines multiple iterables into a single iterable of tuples, allowing you to iterate over multiple sequences simultaneously.

Explanation

It's like matching items from different lists - you take one item from each list and work with them together.

names = ["John", "Jane", "Bob"]
ages = [25, 30, 35]
for name, age in zip(names, ages):
    print(f"{name} is {age} years old")

This pairs each name with its corresponding age.

For Loop with Dictionary

How to iterate through dictionaries?

Definition

For loops can iterate over dictionaries using various methods to access keys, values, or both key-value pairs.

Explanation

It's like going through a dictionary - you can look at just the words, just the definitions, or both together.

person = {"name": "John", "age": 30, "city": "New York"}
for key, value in person.items():
    print(f"{key}: {value}")

This prints each key-value pair in the dictionary.

Practice Exercise: Shopping List Manager

Let's put it all together!

Definition

A shopping list manager that combines for loops, user input, and dictionary operations to create an interactive shopping experience.

Explanation

This exercise combines all the concepts we've learned to create a complete shopping list manager. You'll use for loops to display and process items, dictionaries to store prices, and user input for interaction.

def shopping_list_manager():
    items = {}
    while True:
        print("\n1. Add item")
        print("2. View list")
        print("3. Calculate total")
        print("4. Exit")
        
        choice = input("Enter your choice (1-4): ")
        
        if choice == "1":
            item = input("Enter item name: ")
            price = float(input("Enter item price: "))
            items[item] = price
        elif choice == "2":
            print("\nShopping List:")
            for item, price in items.items():
                print(f"{item}: ${price:.2f}")
        elif choice == "3":
            total = sum(items.values())
            print(f"\nTotal: ${total:.2f}")
        elif choice == "4":
            break

shopping_list_manager()

This exercise combines for loops, dictionaries, and user input to create an interactive shopping list manager.
