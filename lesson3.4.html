Type Conversion

Learn how to convert between different data types in Python.

What is Type Conversion?

Definition and Purpose

Definition

Type conversion is the process of changing the data type of a value to another type, such as converting an integer to a string or a string to a float.

Explanation

Type conversion is useful when you need to perform operations that require values to be of a specific type, like combining numbers or formatting output.

# Example: converting integer to string
age = 25
age_str = str(age)
print("I am " + age_str + " years old.")

This code shows how to convert an integer to a string for concatenation.

Implicit vs. Explicit Conversion

How does Python convert types?

Definition

Implicit conversion is done automatically by Python, while explicit conversion (type casting) is done by the programmer using functions like int(), float(), str(), etc.

Explanation

Python sometimes converts types for you (implicit), but you often need to do it yourself (explicit) to avoid errors.

# Implicit conversion
result = 3 + 2.5  # int + float -> float
print(result)     # 5.5

# Explicit conversion
num_str = "42"
num = int(num_str)
print(num + 8)    # 50

This code shows both implicit and explicit type conversion in Python.

Common Conversion Functions

How to convert between types?

Definition

Python provides built-in functions for converting between types: int(), float(), str(), list(), tuple(), dict(), set(), and more.

Explanation

These functions let you change the type of a value to fit your needs, like turning a string into a number or a list into a set.

# String to int/float
num1 = int("10")
num2 = float("3.14")

# List to set
my_list = [1, 2, 2, 3]
my_set = set(my_list)  # {1, 2, 3}

# Tuple to list
tup = (4, 5, 6)
my_list2 = list(tup)

# List of pairs to dict
pairs = [("a", 1), ("b", 2)]
my_dict = dict(pairs)

This code demonstrates common type conversions in Python.

Converting to String

Why and how to use str()?

Definition

The str() function converts any value to its string representation, which is useful for printing and concatenation.

Explanation

Use str() when you want to display numbers or other types as text.

score = 99
print("Your score is " + str(score))

This code shows how to convert a number to a string for output.

Converting to List, Tuple, Set

How to convert between collections?

Definition

You can convert between list, tuple, and set using list(), tuple(), and set() functions. This is useful for changing mutability or removing duplicates.

Explanation

Convert a tuple to a list to modify it, or a list to a set to remove duplicates.

my_tuple = (1, 2, 3)
my_list = list(my_tuple)
my_list.append(4)

my_list2 = [1, 2, 2, 3]
my_set = set(my_list2)  # {1, 2, 3}

This code shows how to convert between different collection types.

Practice Exercise: Type Converter

Let's put it all together!

Definition

A type converter program that takes user input and converts it to different types.

Explanation

This exercise helps you practice converting between types using Python's built-in functions.

def type_converter():
    value = input("Enter a number: ")
    print("As integer:", int(value))
    print("As float:", float(value))
    print("As string:", str(value))
    print("As list:", list(value))
    print("As set:", set(value))
