Dictionary Methods

Learn how to manipulate and work with dictionaries in Python using built-in methods.

20 minutes Beginner Data Types

What are Dictionary Methods?

Definition and Purpose

Definition

Dictionary methods are built-in functions that perform operations on dictionaries, such as accessing keys, values, or items.

Explanation

Dictionary methods help you manage key-value pairs easily, like listing all keys or finding a value safely.

my_dict = {"name": "Alice", "age": 30}
print(my_dict.keys())  # dict_keys(['name', 'age'])

This code shows how to use a dictionary method to list all keys.

Accessing Keys, Values, and Items

How to access dictionary elements?

Definition

Common methods for accessing elements include keys(), values(), and items().

Explanation

Use keys() to get all keys, values() to get all values, and items() to get all key-value pairs.

my_dict = {"name": "Alice", "age": 30}
print(my_dict.keys())    # dict_keys(['name', 'age'])
print(my_dict.values())  # dict_values(['Alice', 30])
print(my_dict.items())   # dict_items([('name', 'Alice'), ('age', 30)])

This code demonstrates how to access keys, values, and items in a dictionary.

Getting and Setting Values

How to get and set values safely?

Definition

The get() method retrieves a value with a default if the key is not found, and update() adds or modifies key-value pairs.

Explanation

Use get() to avoid errors when a key is missing, and update() to add multiple key-value pairs at once.

my_dict = {"name": "Alice", "age": 30}
print(my_dict.get("email", "Not found"))  # Not found
my_dict.update({"email": "alice@example.com", "city": "London"})
print(my_dict)  # {'name': 'Alice', 'age': 30, 'email': 'alice@example.com', 'city': 'London'}

This code shows how to get and set values in a dictionary safely.

Removing Elements

How to remove elements from a dictionary?

Definition

Common methods for removing elements include pop(), popitem(), and clear().

Explanation

Use pop() to remove a specific key-value pair, popitem() to remove the last inserted item, and clear() to empty the dictionary.

my_dict = {"name": "Alice", "age": 30, "city": "London"}
age = my_dict.pop("age")  # 30, my_dict is now {"name": "Alice", "city": "London"}
item = my_dict.popitem()  # ('city', 'London'), my_dict is now {"name": "Alice"}
my_dict.clear()  # {}

This code shows how to remove elements from a dictionary using different methods.

Copying and Merging

How to copy and merge dictionaries?

Definition

The copy() method creates a shallow copy of a dictionary, and update() merges another dictionary into the current one.

Explanation

Use copy() to duplicate a dictionary, and update() to combine two dictionaries.

dict1 = {"a": 1, "b": 2}
dict2 = dict1.copy()  # {'a': 1, 'b': 2}
dict1.update({"c": 3})  # {'a': 1, 'b': 2, 'c': 3}

This code demonstrates how to copy and merge dictionaries in Python.

Practice Exercise: Dictionary Manager

Let's put it all together!

Definition

A dictionary manager program that uses dictionary methods to add, remove, and display key-value pairs.

Explanation

This exercise combines multiple dictionary methods to create a functional dictionary management tool.

def dictionary_manager():
    my_dict = {}
    while True:
        print("\n1. Add key-value pair")
        print("2. Remove key-value pair")
        print("3. Display dictionary")
        print("4. Exit")
        choice = input("Enter your choice (1-4): ")
        if choice == "1":
            key = input("Enter key: ")
            value = input("Enter value: ")
            my_dict[key] = value
            print("Key-value pair added!")
        elif choice == "2":
            key = input("Enter key to remove: ")
            if key in my_dict:
                my_dict.pop(key)
                print("Key-value pair removed!")
            else:
                print("Key not found!")
        elif choice == "3":
            print("Dictionary:", my_dict)
        elif choice == "4":
            break
        else:
            print("Invalid choice!")
dictionary_manager()

This exercise lets you practice using dictionary methods to manage a dictionary interactively.
