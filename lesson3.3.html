Dictionaries and Sets

Learn about Python's powerful mapping and unique collection types: dictionaries and sets.

20 minutes Beginner Data Types

Dictionaries Basics

What are dictionaries?

Definition

A dictionary is an unordered, mutable collection of key-value pairs. Keys must be unique and immutable. Dictionaries are created using curly braces {}.

Explanation

Think of a dictionary like a real dictionary: you look up a word (key) to get its definition (value).

# Creating a dictionary
person = {"name": "Alice", "age": 30, "city": "London"}

# Accessing values
print(person["name"])  # Alice

# Adding or updating values
person["email"] = "alice@example.com"
person["age"] = 31

# Removing a key-value pair
del person["city"]

# Checking if a key exists
print("email" in person)  # True

This code shows how to create, access, and modify dictionaries in Python.

Dictionary Methods

What can you do with dictionaries?

Definition

Dictionaries have built-in methods for accessing keys, values, items, and for safely retrieving or removing elements.

Explanation

Dictionary methods are like tools for managing your address book: you can list all names, all addresses, or both together.

person = {"name": "Alice", "age": 30, "city": "London"}

# Get all keys, values, and items
print(person.keys())    # dict_keys(['name', 'age', 'city'])
print(person.values())  # dict_values(['Alice', 30, 'London'])
print(person.items())   # dict_items([('name', 'Alice'), ...])

# Get value with default
print(person.get("email", "Not found"))

# Remove and return a value
age = person.pop("age")
print(age)  # 30

# Remove last inserted item
item = person.popitem()
print(item)

This code demonstrates common dictionary methods and their usage.

Sets Basics

What are sets?

Definition

A set is an unordered collection of unique, immutable elements. Sets are created using curly braces {} or the set() function.

Explanation

Think of a set like a bag of unique marbles: no duplicates, and the order doesn't matter.

# Creating sets
colors = {"red", "green", "blue"}
numbers = set([1, 2, 3, 2, 1])  # Duplicates removed

# Adding and removing elements
colors.add("yellow")
colors.remove("red")

# Membership test
print("green" in colors)  # True

This code shows how to create and use sets in Python.

Set Operations

What can you do with sets?

Definition

Sets support mathematical operations like union, intersection, difference, and symmetric difference.

Explanation

Set operations are like combining or comparing groups of friends to see who is in both, only one, or either group.

set1 = {1, 2, 3, 4}
set2 = {3, 4, 5, 6}

# Union
print(set1 | set2)  # {1, 2, 3, 4, 5, 6}

# Intersection
print(set1 & set2)  # {3, 4}

# Difference
print(set1 - set2)  # {1, 2}

# Symmetric difference
print(set1 ^ set2)  # {1, 2, 5, 6}

This code demonstrates set operations in Python.

Dictionaries vs Sets

How are they different?

Definition

Dictionaries store key-value pairs, while sets store only unique values. Both use curly braces, but their purposes and behaviors are different.

Explanation

Think of a dictionary as a labeled box (key-value), and a set as a box of unique items (values only).

# Dictionary: key-value pairs
student = {"name": "Bob", "grade": "A"}

# Set: unique values
unique_grades = {"A", "B", "C", "A"}  # {'A', 'B', 'C'}

Practice Exercise: Student Directory

Let's put it all together!

Definition

A student directory that uses a dictionary to store student information and a set to track unique courses.

Explanation

This exercise combines dictionaries and sets to manage student data and ensure unique course names.

def student_directory():
    students = {}
    courses = set()
    while True:
        print("\n1. Add student")
        print("2. View students")
        print("3. Add course")
        print("4. View courses")
        print("5. Exit")
        choice = input("Enter your choice (1-5): ")
        if choice == "1":
            name = input("Enter student name: ")
            age = input("Enter age: ")
            students[name] = {"age": age}
            print("Student added!")
        elif choice == "2":
            if not students:
                print("No students!")
            else:
                for name, info in students.items():
                    print(f"{name}: {info}")
        elif choice == "3":
            course = input("Enter course name: ")
            courses.add(course)
            print("Course added!")
        elif choice == "4":
            print("Courses:", courses)
        elif choice == "5":
            break
        else:
            print("Invalid choice!")
student_directory()

This exercise combines dictionaries and sets to create a simple student directory and course tracker.
