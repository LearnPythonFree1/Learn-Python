Question 1
What are strings in Python and how do you create them?
Answer
Strings in Python are sequences of characters enclosed in quotes. You can create strings using: 1. Single quotes: name = 'Python' 2. Double quotes: message = "Hello, World!" 3. Triple quotes (for multi-line strings): text = '''This is a multi-line string''' 4. String concatenation: first = "Hello" last = "World" greeting = first + " " + last 5. String repetition: pattern = "=" * 10 # Creates "=========="
Definition
Strings are immutable sequences of Unicode characters in Python. They are one of the most commonly used data types and come with a rich set of built-in methods for manipulation.
Question 2
What are the most commonly used string methods in Python?
Answer
Common string methods include: 1. Case manipulation: text = "Hello World" text.upper() # "HELLO WORLD" text.lower() # "hello world" text.title() # "Hello World" text.capitalize() # "Hello world" 2. Searching and replacing: text = "Hello World" text.find("World") # Returns 6 text.replace("World", "Python") # "Hello Python" text.count("l") # Returns 3 3. Stripping whitespace: text = " Hello " text.strip() # "Hello" text.lstrip() # "Hello " text.rstrip() # " Hello" 4. Splitting and joining: text = "Hello,World,Python" text.split(",") # ["Hello", "World", "Python"] "-".join(["Hello", "World"]) # "Hello-World"
Definition
String methods are built-in functions that allow you to manipulate strings in various ways. They return new strings (since strings are immutable) and don't modify the original string.
Question 3
What are the different ways to format strings in Python?
Answer
Python offers several string formatting methods: 1. f-strings (Python 3.6+): name = "Alice" age = 25 message = f"{name} is {age} years old" 2. str.format(): message = "{} is {} years old".format(name, age) message = "{n} is {a} years old".format(n=name, a=age) 3. % operator (old style): message = "%s is %d years old" % (name, age) 4. Template strings: from string import Template template = Template("$name is $age years old") message = template.substitute(name=name, age=age)
Definition
String formatting allows you to create dynamic strings by inserting values into placeholders. f-strings are the most modern and readable approach, while other methods are maintained for compatibility.
Question 4
What are the common string operations in Python?
Answer
Common string operations include: 1. Indexing and slicing: text = "Python" text[0] # 'P' text[-1] # 'n' text[0:3] # 'Pyt' text[::2] # 'Pto' 2. Length and membership: len(text) # 6 'P' in text # True 'x' in text # False 3. String comparison: "Python" == "python" # False "Python" < "python" # True (case-sensitive) 4. String methods for checking: text.isalpha() # True text.isdigit() # False text.startswith("Py") # True text.endswith("on") # True
Definition
String operations allow you to access, manipulate, and analyze string content. They include indexing, slicing, length checking, membership testing, and various string methods for content validation.
Question 5
How do you handle special characters and escaping in Python strings?
Answer
Python provides several ways to handle special characters: 1. Escape sequences: text = "Line 1\nLine 2" # New line text = "Tab\there" # Tab text = "Quote: \"Hello\"" # Quote text = "Backslash: \\" # Backslash 2. Raw strings (r-strings): path = r"C:\Users\Name\Documents" # No need to escape backslashes 3. Triple quotes for multi-line: text = """Line 1 Line 2 Line 3""" 4. Unicode escape: text = "\u0041" # 'A' text = "\U0001F600" # '😀'
Definition
String escaping allows you to include special characters in strings that would otherwise be difficult to represent. Python provides various escape sequences and special string prefixes to handle these cases.
